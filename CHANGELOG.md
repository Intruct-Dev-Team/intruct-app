# Changelog

## [Latest] - Обновление дизайна настроек

### Изменено

- ✅ Убраны тени со всех компонентов (Stats, Course, Create Course cards)
- ✅ Переделана страница настроек в стиле iOS
- ✅ Заменен выбор темы с карточек на Switch переключатель
- ✅ Добавлено модальное окно для выбора языка
- ✅ Улучшена структура компонентов настроек

### Новые компоненты

- `SettingsCard` - карточка-контейнер для настроек
- `SettingsItem` - элемент настройки с иконкой, заголовком и описанием
- `ThemeToggle` - Switch для переключения темной темы
- `LanguageModal` - Sheet модальное окно для выбора языка

### Удалено

- `ThemeSelector` - заменен на ThemeToggle
- `LanguageSelector` - заменен на LanguageModal
- `SettingsSection` - заменен на SettingsCard

### Дизайн

- Настройки теперь в едином белом блоке (как на скриншоте)
- Dark Mode с переключателем справа
- Language с стрелкой и модальным окном
- Разделитель между элементами настроек
- Иконки в серых квадратах слева
