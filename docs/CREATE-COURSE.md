# Create Course Flow

## Обзор

Функционал создания курса реализован как многошаговый процесс с использованием AI для генерации контента.

## Структура

### Экран создания курса (`app/create-course.tsx`)

Главный экран с навигацией между шагами:

- Индикатор прогресса (Step 1 of 3)
- Кнопки навигации (Back / Next)
- Динамическое отображение шагов

### Шаги создания

#### 1. Attach Materials (`attach-materials-step.tsx`)

- Загрузка файлов (PDF, DOC, TXT, Images)
- Добавление ссылок на материалы
- Drag & drop интерфейс

#### 2. Course Details (`course-details-step.tsx`)

- Название курса
- Описание курса
- Валидация полей

#### 3. Review & Create (`review-step.tsx`)

- Просмотр всех данных
- Количество файлов и ссылок
- Информация о процессе создания

### Модальное окно создания (`creating-course-modal.tsx`)

Показывает прогресс создания курса:

- **Processing materials** - анализ загруженных материалов
- **Creating course plan** - создание структуры курса
- **Writing lessons** - генерация уроков

**Особенности:**

- Анимированный прогресс-бар
- Индикаторы для каждого шага
- Автоматическое закрытие после завершения
- Spinner во время процесса
- Галочка при завершении

## Компоненты

### StepIndicator

Визуальный индикатор текущего шага:

- Прогресс-бары для каждого шага
- Текст "Step X of Y"
- Цветовая индикация (активный/завершенный)

## Процесс создания

1. Пользователь нажимает "Create Course" на главной странице
2. Открывается экран с шагом 1 (Attach Materials)
3. Пользователь загружает файлы и добавляет ссылки
4. Нажимает "Next" → переход на шаг 2 (Course Details)
5. Заполняет название и описание
6. Нажимает "Next" → переход на шаг 3 (Review)
7. Проверяет данные и нажимает "Create Course"
8. Открывается модальное окно с прогрессом
9. После завершения (~6 секунд) автоматически закрывается
10. Пользователь возвращается на главную страницу
